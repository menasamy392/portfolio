# Mina Samy Abood — Cybersecurity Portfolio

A modern, hacker-themed personal portfolio website.

## Structure
```
portfolio/
├── index.html       → Main HTML file (open this to run the site)
├── style.css        → All styles (dark hacker theme)
├── script.js        → Animations, interactivity, matrix rain
└── assets/
    ├── images/      → Place profile images here
    └── icons/       → Place custom icons here
```

## Features
- Matrix rain background animation
- Glitch effect hero name
- Typing title animation  
- Animated skill bars
- Scroll-triggered reveals
- Orbital tool badges
- Interactive terminal UI
- Fully responsive (mobile-first)
- Smooth navigation

## How to Run
Simply open `index.html` in any modern browser — no server required.
